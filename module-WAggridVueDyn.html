Events

Name Payload Type Description
refresh Null 指調用組件的method，無輸入與回傳，刷新表格
showKeys Array 指調用組件的method，輸入指定欲顯示欄位的keys，數量最多為原本初始化的keys，可更改順序，無回傳
setHeadFilter Array 指調用組件的method，輸入指定欄位的key與要過濾的值value，會於界面上指定欄進行過濾
clearHeadFilter Array 指調用組件的method，輸入指定欄位的key並清除當前所使用的過濾值
clearHeadFilterAll Array 指調用組件的method，無輸入與回傳，清除當前所有欄位所使用的過濾值
getDisplayData Null 指調用組件的method，無輸入，會回傳目前表格所顯示之數據
getInstance Null 指調用組件的method，無輸入，會回傳ag-grid表格實例物件
fitColumns Null 指調用組件的method，無輸入與回傳，自動調整欄寬
pasteText String 指調用組件的method，輸入text為欲貼上的文字，showRowIndNow為貼上的列指標整數，預設null，showColKeyNow為貼上的欄位key值字串，預設null
downloadDisplayData Null 指調用組件的method，無輸入，下載目前表格所顯示之數據為Excel檔，亦會回傳目前表格所顯示之數據
downloadData Null 指調用組件的method，無輸入，下載數據為Excel檔，亦會回傳數據
uploadData Object 指調用組件的method，輸入設定物件，其中pathItems代表調用wsemi的getDataFromExcelFileU8ArrDyn所傳入的xlsx的來源網址陣列，beforeUpload代表上傳前的處理數據函數，parseSheetInd代表提取Excel檔案的第幾個sheet整數，預設為0。呼叫時會開啟選擇檔案視窗給使用者上傳Excel檔案，需為含有標題的Excel檔，標題需為各欄位key值而非head。回傳為Promise，resolve為成功上傳之數據，reject為失敗原因
