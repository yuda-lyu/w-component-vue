WDrawer

Description:

Props

Name Type Default value Required? Description
value Boolean false No 輸入是否顯示布林值，預設false
mode String 'from-left' No 輸入抽屜顯示位置字串，可使用'from-left'與'from-right'，預設'from-left'
drawerWidth Number 200 No 輸入抽屜寬度數字，單位為px，預設200
drawerWidthMin Number null No 輸入使用拖曳抽屜寬度分隔條(dragDrawerWidth=true)時，拖曳抽屜寬度分隔條最小值數字，預設null
drawerWidthMax Number null No 輸入使用拖曳抽屜寬度分隔條(dragDrawerWidth=true)時，拖曳抽屜寬度分隔條最大值數字，預設null
afloat Boolean false No 輸入是否為浮動顯示布林值，設為true時浮在內容區上故不壓縮內容區寬度，預設false
afloatByFix Boolean false No 輸入浮動顯示時是否使用fixed布林值，若為true使用'fixed'反之使用'absolute'，預設false
overlayOpacity Number 0.45 No 輸入浮動顯示時抽屜外側陰影層之透明度數字，預設0.45
overlayColor String 'grey darken-2' No 輸入浮動顯示時抽屜外側陰影層背景顏色字串，預設'grey darken-2'
drawerZIndex Number 1000 No 輸入浮動顯示時抽屜使用z-index數字，預設1000
dragDrawerWidth Boolean false No 輸入是否使用拖曳抽屜寬度分隔條布林值，預設false
drawerBarColor String '#ddd' No 輸入分隔條顏色字串，預設'#ddd'
drawerBarSize Number 2 No 輸入分隔條尺寸數字，為分隔條寬度，單位為px，預設2
drawerBarBorderColor String 'transparent' No 輸入分隔條框線顏色字串，預設'transparent'
drawerBarBorderSize Number 3 No 輸入分隔條框線寬度數字，單位為px，預設3，通常配合barBorderColor='transparent'使可拖曳區加大又不遮蔽可視區
autoSwitch Boolean false No 輸入是否自動顯隱布林值，設為true時當組件寬度過小時則隱藏抽屜且改為浮動顯示，反之則恢復顯示且改為佔版顯示，預設false
switchWidth Number 340 No 輸入當自動顯隱時判斷組件寬度過小之門檻寬度數字，單位為px，預設340
